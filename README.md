# MAI-IR — Информационный поиск

Проект по курсу **«Информационный поиск»**.  
Тема корпуса: **Machine Learning (arXiv, категория cs.LG)**.

Реализовано:
- сбор и валидация корпуса (metadata → pdf → txt);
- токенизация (C++);
- стемминг (C++);
- частотный анализ и подготовка данных для закона Ципфа (C++);
- построение булевого (инвертированного) индекса (C++);
- булев поиск с `AND / OR / NOT` и скобками (C++);
- CLI и Web-интерфейс (Flask).

---

## Структура репозитория

- `lab1/` — сбор корпуса + проверки и статистика (`verify_corpus.py`, `compute_stats.py`)
- `lab2/` — обёртка-пайплайн (pipeline) для воспроизводимого запуска
- `lab3/` — токенизация (CMake → `tokenize`)
- `lab4/` — стемминг (CMake → `stem`)
- `lab5/` — закон Ципфа / частоты (CMake → `zipf`)
- `lab7/` — построение индекса (CMake → `indexer`)
- `lab8/` — булев поиск (CMake → `search`) + `webapp.py`
- `scripts/` — вспомогательные скрипты (сборка индекса для веба)
- `corpus_sample/` — небольшой подкорпус для демонстрации (корпус полностью не коммитится)
- `report/` — отчёт (LaTeX) и рисунки

---

## Требования

### C++ часть
- macOS / Linux
- `cmake` ≥ 3.16
- компилятор с C++17 (AppleClang / clang / g++)

### Python часть
- Python 3.9+
- зависимости для Lab1/Lab2 и Web:
  - `flask` (для `lab8/webapp.py`)
